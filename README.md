This project is based on the LLaVA-NeXT-Video codebase, with several fine-tuning modifications tailored to our lab’s self-created virtual reality (VR) dataset. The code was developed as part of my final year thesis at The University of Hong Kong (HKU), focusing on:

**Video Understanding on a Self-Created Virtual Reality Dataset Using Vision-Language Models**

I adapt large vision-language models (VLMs) to the VR setting, addressing domain-specific challenges such as motion-rich scenes, synthetic environments, and fine-grained human actions.
